Starting With Emmet: 20-Jan-2023

Adding Class and Id attribute

  • Class

    Example: p.Classname

    
                <p class="classname"></p>
  • Id

    Example: p#Idname

    
                <p id="Idname"></p>
  • Multiple class

    Example: p.Class1.Class2

    
                <p class="Class1 Class2"></p>

Adding Custom attributes

Example: img[src='image.jpg']


        <img src="image.jpg" alt="">

Adding Text

Example: p{This is a paragraph}


        <p>This is a paragraph</p>

Parent-Child Grouping

Example: div>p>ul>li


        <div>
            <p>
                <ul>
                    <li></li>
                </ul>
            </p>
        </div>

Adding Sibling

Example: div>p+p


        <div>
            <p></p>
            <p></p>
        </div>

Multiplication

Example: div>ul>li*5


        <div>
            <ul>
                <li></li>
                <li></li>
                <li></li>
                <li></li>
                <li></li>
            </ul>
        </div>
